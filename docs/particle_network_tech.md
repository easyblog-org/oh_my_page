# AI IDE 指令：生成粒子网络球组件（ParticleNetworkSphere）

## 任务目标
基于现有项目结构（React + TypeScript + Three.js，组件遵循 `HeroAnimationProps` 接口），生成一个 **粒子网络球体** 组件，用于替换 Hero 区域右侧的动画图形。该组件应展示一个由粒子点组成的球体，粒子之间根据距离产生连线，整体缓慢旋转，颜色使用主色 `#3b82f6`，背景透明，具有科技感和数据流动的寓意。

## 技术要求
- 组件名：`ParticleNetworkSphere.tsx`
- 接收 `width`, `height`, `className` 作为 props（与 `HeroAnimationProps` 一致）。
- 仅在桌面端（`window.innerWidth >= 768`）加载，移动端返回 `null` 或简单降级。
- 使用 Three.js 动态导入（`import("three")`），避免增加打包体积。
- 粒子数量：500～800 个，均匀分布在球面上。
- 每个粒子为一个小的圆形点（`PointsMaterial` 或 `SphereGeometry` 小方块，推荐 `Points` + 圆形纹理）。
- 连线：计算粒子之间的距离，若小于阈值（如半径的 0.3 倍），则用 `LineSegments` 绘制线条，线条半透明、随距离改变透明度或颜色。
- 动画：整个球体绕 Y 轴缓慢自转，同时可绕 X 轴轻微摆动。
- 光照：添加环境光和点光源，使球体有立体感（如果使用 `Points` 则不需要光照，但为了连线效果可加）。
- 清理：在组件卸载时销毁所有 Three.js 资源（几何体、材质、渲染器等）。
- 性能：粒子数量适中，连线数量控制在合理范围（可使用空间分割优化，或仅预计算一部分连线）。
- 颜色：粒子颜色为主色 `#3b82f6` 及其变体（如亮蓝 `#60a5fa`），连线颜色同主色，透明度 0.3～0.6。
- 背景透明，与深色渐变背景融合。

## 可选增强
- 粒子大小随到相机的距离变化（透视效果）。
- 连线在粒子附近产生微弱的流光效果（通过纹理动画或顶点颜色）。
- 鼠标悬浮时球体转速加快或高亮。

## 代码输出要求
请输出完整的 `ParticleNetworkSphere.tsx` 文件代码，包含：
- 必要的 import（React, useRef, useEffect, 类型声明）
- 组件定义，使用 `useEffect` 初始化 Three.js
- 粒子位置计算（球面均匀分布，可使用正态分布或 Fibonacci 算法）
- 连线生成逻辑（双重循环，性能优化：使用空间索引或限制最大连线数）
- 动画循环（`requestAnimationFrame`）
- 窗口尺寸适配
- 清理函数
- 移动端判断与降级（返回 `null` 或简单 div）

## 样式要求
- 容器使用 `hidden md:flex` 类，尺寸由父组件传入的 `width/height` 决定。
- 背景透明，画布无边框无阴影。
