# AI IDE 指令：实现动态光柱/脉冲波组件（Three.js）

## 任务目标
为个人作品集网站的 Hero 区域右侧，创建一个 **动态光柱/脉冲波** 组件，替代原有的莫比乌斯环。视觉效果：从中心向外不断扩散的半透明环形波，多层叠加，像雷达脉冲或能量波，颜色使用主色 `#3b82f6`，背景透明，风格极简、科技感强。

## 技术要求
- 使用 **React + TypeScript + Three.js**（无需额外 UI 库）。
- 组件名为 `PulseRing.tsx`，放置在 `src/components/` 目录下。
- 仅当屏幕宽度 ≥ 768px 时渲染（移动端不加载，避免性能浪费）。
- 动画：至少 3~5 个环形波，从中心（0,0,0）以不同速度向外扩散，到达最大半径后透明度归零并重置到中心，形成无限循环。环的半径从 0.5 到 2.5 之间变化，厚度极薄（0.02~0.05）。
- 环的材质：`MeshBasicMaterial` 或 `MeshStandardMaterial` 加 `emissive` 属性，使用半透明和自发光效果，颜色 `#3b82f6`，透明度随半径增大而减小（从 0.8 到 0）。
- 相机：透视相机，位置 (0, 1.5, 6)，视角能完整看到半径 3 以内的范围。
- 场景背景透明，与深色渐变背景融合。
- 性能：使用 `requestAnimationFrame`，在组件卸载时清理资源（`dispose` 几何体、材质、渲染器）。

## 可选增强（加分项）
- 环的截面可以略带宽高（使用圆环几何体 `TorusGeometry`，小管径）。
- 添加一个微弱的中心球体或点光源。
- 环的颜色可以随时间略微变化（Hue 偏移）或添加渐变纹理。

## 集成方式
- 在 `Hero.tsx` 中，右侧容器位置（原 `ThreeScene` 处）引入 `<PulseRing />`。
- 父容器需有明确的宽高（例如 `w-full h-full min-h-[400px]`），确保 canvas 正确显示。

## 代码输出要求
请输出完整的 `PulseRing.tsx` 文件内容，包含：
- 必要的 import（React, useEffect, useRef, THREE）
- 组件定义
- 所有样式、动画逻辑、清理逻辑
- 移动端判断与条件返回（返回空 div 或 null）
- 注释清晰，说明动画原理。

用户打开网站后，右侧看到一个由中心向外不断扩散的蓝色光环，三层光环依次出现、扩大、消失，循环往复，整体缓慢自转（可选），给人一种“技术能量正在释放”的感觉。
